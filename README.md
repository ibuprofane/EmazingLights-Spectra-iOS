# EmazingLights Spectra Evolution Bluetooth Glove Set App

This repo includes the original iOS source code for the EmazingLights Spectra light gloves. It was written from 2015 to 2016 using a now obsolete version of Swift and **does not compile** with modern versions of Xcode. The code is being made available so that legacy hardware owners can adapt the communication layer to create new tools to configure their lights.
